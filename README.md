# Rebbit 🐸

A premium browser extension that provides a visual overhaul and power-user features for Reddit. Developed by **duckbox**.

## ✨ Features
- **Visual Overhaul:** Sleek, modern redesign of the Reddit interface.
- **Multiple Layouts:** Wide Mode, Compact Mode, and Focus Mode.
- **Custom Accent Colors:** Make it yours with custom color themes.
- **Comment Enhancements:** Colored depth lines and auto-collapse for bot comments.
- **Keyboard Shortcuts:** Fast navigation (J/K), upvote/downvote (A/Z), and a Command Palette (Ctrl+K).
- **Undertale Theme:** A retro, pixel-art aesthetic for fans of classic RPGs.

### Manual Installation (Developer Mode)
1. Clone this repository or download the ZIP file.
2. Open your browser's extension management page (`chrome://extensions` or `edge://extensions`).
3. Enable **Developer mode**.
4. Click **Load unpacked** and select the directory containing the extension files.
